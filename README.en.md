## About

This project provides a Korean translation of **CS146S: The Modern Software Developer** from Stanford University. Our goal is to deliver AI-era software development methodologies to Korean developers.

### Authorization

This translation is officially authorized by **Mihail Eric**, the original course creator:

> "Thanks for the kind words Goobong. Super thing feel free to translate the materials to Korean, as long as there is proper attribution to the source materials. If helpful, I can also host the korean on a separate subdomain. kr.themodernsoftware.dev"
>
> — Mihail Eric, 2026-01-03

## Development Setup

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev

# Production build
pnpm build

# Preview build
pnpm preview
```
