## 개발 환경 설정

```bash
# 의존성 설치
pnpm install

# 개발 서버 실행
pnpm dev

# 프로덕션 빌드
pnpm build

# 빌드 결과 미리보기
pnpm preview
```

## 프로젝트 구조

```
stanford-cs146s-kr/
├── src/
│   ├── components/    # React 컴포넌트
│   ├── pages/         # 페이지 컴포넌트
│   ├── content/       # 커리큘럼, FAQ 데이터
│   └── main.tsx       # 앱 엔트리
├── public/            # 정적 파일
└── dist/              # 빌드 결과
```
